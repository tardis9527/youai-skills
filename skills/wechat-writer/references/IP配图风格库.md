# IP 配图风格库

> 阶段 8 第三步使用。这是一套**带 IP 感**的配图风格资产库：每个风格都有一个**固定出场的 IP 角色**，
> 让配图不再是"配一张好看的图"，而是**用一个可识别的角色去承担画面的核心动作**，从而获得跨图、跨篇的记忆点。
>
> 用法：阶段 8 里，先按本文的写作风格 / 内容调性 / 目标读者，用「一、选择指引」推荐 2–4 个候选 IP（含兜底），
> 让用户选一套；再把选定 IP 的「角色锚定描述」逐字嵌进每条出图提示词（见 `配图设计与生图提示词.md` 第四节），
> 保证同一角色在多张图里长得一样。**本库只提供风格与角色规格，不调用任何生图接口。**

---

## 一、选择指引：先按内容和读者推荐，别让用户从零想

不要问用户"你想要什么风格"，而是**从内容本身和读者推导**出 2–4 个候选，附一句话理由让用户挑。

| 本文写作风格 / 内容调性 | 首选 IP | 备选 IP | 兜底 |
|---|---|---|---|
| 科技速报（产品/资讯/AI） | 墨点 | 方头 | 纪实摄影 |
| 认知深度（方法论/思维模型） | 墨点 | 砚生 | —— |
| 犀利点评（锐评/吐槽/热点） | 阿波 | 墨点 | —— |
| 故事叙事（经历/复盘/人物） | 灯灯 | 暖团 | 纪实摄影 |
| 干货教程（步骤/流程/B端） | 方头 | 墨点 | —— |
| 情感 / 生活 / 成长 / 治愈 | 暖团 | 灯灯 | —— |
| 文化 / 历史 / 国风 / 慢生活 | 砚生 | 灯灯 | —— |
| 硬新闻 / 严肃财经 / 需强真实可信 | 纪实摄影 | —— | —— |

> 三条挑选提醒：
> 1. **读者优先于题材**：同一题材，面向开发者用「墨点」，面向大众用「暖团」，面向 Z 世代用「阿波」。
> 2. **一篇文章原则上只用一个 IP**（封面 + 文中插图统一角色），除非用户明确要多角色对比。
> 3. **用户偏好会进知识库**：阶段 0 若加载到用户"偏好的配图 IP 风格"，把它放进候选并标注"你上次用的"。

---

## 二、6 个固定 IP 风格

每个 IP 的字段统一为：**定位 / 角色人设 / 视觉 DNA / 适配场景 / 适配读者 / 在画面里做什么 / 禁忌 / 角色锚定描述 / 提示词片段示例**。
其中「角色锚定描述」是一段**要逐字复制进每条提示词**的固定文本，是跨图一致的关键。

---

### IP-1 · 墨点（Mò Diǎn）— 极简手绘怪诞

- **定位**：在白纸上一本正经做荒诞小事的黑色小怪，把抽象判断变成看得见的物理动作。
- **角色人设**：黑色实心小豆子身体，白色圆点眼睛，细腿偶有细胳膊，空表情、冷静认真；像个低调的系统操作员，冷幽默、不卖萌。
- **视觉 DNA**：纯白背景（无纸纹/阴影/渐变）；黑色手绘细线、轻微抖动；大量留白，主体占 40%–60%；
  少量手写批注——**橙色**=主路径/箭头/流向，**红色**=重点/问题/结果，**蓝色**=补充/系统状态（可不用）。
- **适配场景**：AI / 产品 / 工作流 / 效率工具 / 方法论 / 认知升级。
- **适配读者**：开发者、产品经理、独立创造者、知识工作者。
- **在画面里做什么**：拉线汇聚信息、卡在流程断点里、扳动"判断"杆、变成筛选漏斗、把想法塞进怪装置——**承担核心动作，不站在旁边看**。
- **禁忌**：不可爱化 / 不商业插画 / 不 PPT 信息图 / 不复杂背景 / 不在左上角写"流程图/架构图"标题。
- **角色锚定描述（逐字嵌入）**：
  > 画面里的固定小角色叫"墨点"：一个纯黑色实心、白色圆点眼睛、细腿、无表情、手绘轮廓略微抖动的小怪物，认真而略显荒诞地操作着画面里的东西。
- **提示词片段示例**：`纯白背景手绘线稿风格，黑色抖动细线，大量留白，少量橙红蓝手写中文批注；墨点正在把一团"想法"塞进一台怪机器；避免：卡通感、PPT 感、复杂背景、文字乱码。`

### IP-2 · 暖团（Nuǎn Tuán）— 治愈手绘水彩

- **定位**：陪你坐一会儿的奶油色小团子，给读者"被轻轻抱一下"的温度。
- **角色人设**：奶油/米白色圆润软身体，两点腮红，眯眼温和微笑，短手短脚；性格温柔、松弛、陪伴，不说教。
- **视觉 DNA**：米白/奶油背景；柔和水彩晕染、圆润线条、轻微纸纹与柔光；暖黄打底 + 莫兰迪低饱和点缀；整体软、静、暖。
- **适配场景**：情感 / 日常生活 / 个人成长 / 治愈 / 复盘 / 自我关怀 / 睡前读物。
- **适配读者**：泛大众、女性生活方式读者、情感共鸣型、都市疲惫人群。
- **在画面里做什么**：抱着一颗心/一本书、窝在角落、给读者递一杯热饮、陪着一株小植物——用姿态传递陪伴与松弛。
- **禁忌**：不锐利 / 不冷调 / 不高饱和 / 不商业广告感 / 不夸张搞怪。
- **角色锚定描述（逐字嵌入）**：
  > 画面里的固定小角色叫"暖团"：一个奶油米白色、圆润柔软、两点腮红、眯眼温和微笑、短手短脚的手绘小团子，动作轻柔、气质松弛温暖。
- **提示词片段示例**：`米白色背景水彩晕染风格，圆润柔和线条，暖黄与莫兰迪低饱和色，轻微纸纹柔光；暖团窝在窗边抱着一杯热茶陪着读者；避免：锐利线条、高饱和、广告感、冷色调。`

### IP-3 · 方头（Fāng Tóu）— 扁平几何商务

- **定位**：条理清楚、把复杂事拆明白的几何化职场操作员。
- **角色人设**：方头方脑，圆角矩形身体，纯色块填充，极简圆点五官；性格高效、有条理、靠谱、不花哨。
- **视觉 DNA**：干净白/浅灰背景；扁平色块、清晰层次；**一主色 + 一辅色**（克制的品牌色，如藏蓝+橙）；几何图形（方块/箭头/网格）；细描边或无描边。
- **适配场景**：职场 / 商业 / 干货教程 / SaaS / B 端产品 / 效率方法 / 流程与框架拆解。
- **适配读者**：职场白领、企业管理者、B 端从业者、专业人群。
- **在画面里做什么**：搭建结构、拆解流程、搬运模块、指示路径——让画面的结构一眼看懂。
- **禁忌**：不杂乱 / 不高饱和撞色 / 不手绘毛边 / 不卖萌 / 不堆满图标。
- **角色锚定描述（逐字嵌入）**：
  > 画面里的固定小角色叫"方头"：一个方头方脑、圆角矩形身体、纯色块填充、极简圆点五官的扁平几何小人，动作利落、气质专业可靠。
- **提示词片段示例**：`干净浅灰背景扁平几何风格，藏蓝为主橙色为辅，色块清晰层次分明；方头正在把三个模块搭成一条流程；避免：杂乱、高饱和撞色、手绘毛边、图标堆砌。`

### IP-4 · 阿波（Ā Bō）— 波普涂鸦吐槽

- **定位**：自带弹幕、敢说敢炸、一眼有态度的涂鸦小人。
- **角色人设**：粗黑描边，大圆眼，咧嘴，表情和动作都夸张，常配对话气泡；性格敢说、浮夸、有梗、不端着。
- **视觉 DNA**：高饱和撞色（明黄/正红/亮蓝/黑）；粗黑描边；漫画拟声词、速度线、闪线、网点；贴纸/表情包质感但克制不廉价。
- **适配场景**：热点锐评 / 犀利吐槽 / 潮流 / 娱乐 / 反差观点 / 社会话题。
- **适配读者**：Z 世代、年轻潮流人群、追热点的读者。
- **在画面里做什么**：举牌喊话、瞪眼、摔倒、被"雷"劈、翻白眼——把文章的情绪和态度直接演出来。
- **禁忌**：不低饱和 / 不性冷淡 / 不精致写实 / 不温吞 / 不过度用梗到低俗。
- **角色锚定描述（逐字嵌入）**：
  > 画面里的固定小角色叫"阿波"：一个粗黑描边、大圆眼、咧嘴、表情夸张、动作浮夸的波普涂鸦小人，情绪外放、自带梗感。
- **提示词片段示例**：`高饱和撞色波普涂鸦风格，粗黑描边，明黄正红亮蓝，配速度线和拟声词；阿波举着牌子瞪大眼睛吐槽；避免：低饱和、写实精修、性冷淡、温吞。`

### IP-5 · 砚生（Yàn Shēng）— 新中式水墨

- **定位**：一笔带过、意到笔不到的水墨小书童，营造东方留白意境。
- **角色人设**：水墨写意飞白笔触勾出的小书童，身形简练、留白处理，或换成一只水墨小兽；性格内敛、雅致、有出处、慢。
- **视觉 DNA**：宣纸/米白背景；水墨浓淡枯润；一抹**朱砂红**或**石青蓝**点缀；飞白与晕染；大量留白；竖版尤其好看。
- **适配场景**：文化 / 历史 / 国风 / 茶酒 / 传统智慧 / 慢生活 / 东方调性品牌。
- **适配读者**：文化爱好者、国风受众、高客单价品牌读者、中年高知。
- **在画面里做什么**：提笔、观景、煮茶、独行、临窗——用意境陪衬观点，而非解释说明。
- **禁忌**：不高饱和 / 不赛博科技感 / 不卡通 / 不把画面塞满 / 不用现代硬边图形。
- **角色锚定描述（逐字嵌入）**：
  > 画面里的固定小角色叫"砚生"：一个用水墨写意飞白笔触勾出、身形简练留白、气质内敛雅致的小书童，动作从容、意到笔不到。
- **提示词片段示例**：`宣纸米白背景新中式水墨风格，水墨浓淡枯润，飞白晕染，一抹朱砂点缀，大量留白；砚生临窗煮茶远望；竖版；避免：高饱和、赛博感、卡通、画面拥挤。`

### IP-6 · 灯灯（Dēng Dēng）— 复古胶片故事

- **定位**：黄昏里陪你回忆的怀旧剪影，一个"讲故事的人"。
- **角色人设**：戴围巾、背布包、逆光剪影的怀旧小人，或换成一盏拟人的暖黄小台灯；性格念旧、克制、有故事。
- **视觉 DNA**：暖黄偏色；胶片颗粒；漏光；老照片边框/圆角；低对比；黄昏与夜色调。
- **适配场景**：个人经历 / 深度故事 / 城市夜归 / 情怀 / 品牌故事 / 人物专访。
- **适配读者**：30+ 都市人、文艺读者、情感共鸣型受众。
- **在画面里做什么**：走在路上、望向窗外、翻看旧物、在夜里亮着一盏灯——以"陪伴讲述者"的姿态出现。
- **禁忌**：不高清锐利 / 不冷调 / 不夸张 / 不商业硬广 / 不用现代扁平图形。
- **角色锚定描述（逐字嵌入）**：
  > 画面里的固定小角色叫"灯灯"：一个戴围巾、背布包、逆光剪影的怀旧小人（或一盏拟人的暖黄小台灯），气质念旧克制，像一个安静的讲故事的人。
- **提示词片段示例**：`暖黄偏色复古胶片风格，胶片颗粒与漏光，低对比黄昏色调，老照片边框；灯灯背着布包走在傍晚的街道回头望；避免：高清锐利、冷色调、夸张、商业硬广。`

## 三、2 个兜底方案（IP 不合适时用）

### 兜底 A · 纪实摄影感（无固定 IP）
- **什么时候用**：硬新闻、严肃财经、政策解读、需要强真实可信度的内容——**上一个卡通 IP 反而显得不专业**时。
- **做法**：真实光影、纪实构图、浅景深、克制色调；封面为标题预留留白区。不放 IP 角色。
- **一句话理由模板**：这篇偏严肃/需要真实感，用纪实摄影比卡通 IP 更稳、更可信。

### 兜底 B · 自由融合 / 另拟
- **什么时候用**：用户想要两套 IP 的融合（如"暖团的温度 + 方头的条理"），或希望完全另拟一个新方向。
- **做法**：明确融合的取舍（谁定色调、谁定角色、谁定构图），或按用户口味现拟一套；满意后可按「五、可扩展」存回本库。

## 四、IP 一致性技巧（多张图/多篇文章长得一样）

只出提示词、不生图的前提下，靠这几招把角色"锚"住：

1. **锚定描述逐字复用**：同一篇文章的每张图，都原样粘贴该 IP 的「角色锚定描述」，一个字都不改。
2. **固定 seed / 种子**：在支持的工具里（MJ 的 `--seed`、SD 的 Seed）用同一个种子值生成同一角色。
3. **首图当参考图**：先定一张满意的，再用 image-to-image / 垫图 / 角色参考（如 MJ `--cref`）生成后续图。
4. **只改场景不改角色**：跨图之间只替换"它在做什么/在哪"，角色外形描述保持完全一致。
5. **色彩语义一致**：同一 IP 的主色/点缀色在全篇统一（如墨点全篇只用橙红蓝三色批注）。

> 把以上第 2–3 点作为"给用户的操作建议"写进交付说明，帮用户在自己的生图工具里稳住角色。

## 五、可扩展：把新 IP 存回本库

如果用户提供了一个自有吉祥物/品牌形象，或在一次创作中拟出了满意的新 IP，可仿照「二、」的字段格式，
新增一个 IP 条目写回本文件（定位 / 角色人设 / 视觉 DNA / 适配场景 / 适配读者 / 动作 / 禁忌 / 角色锚定描述 / 提示词片段），
以后即可像内置 IP 一样复用——这与写作风格库 `references/styles/` 的可扩展逻辑一致。
