# 配图设计与生图提示词

> 阶段 6 使用。基于定稿文案，以资深视觉设计师视角设计配图方案，产出可直接粘贴到任意 AI 生图工具（即梦 / Midjourney / Stable Diffusion / GPT-Image 等）使用的中文出图提示词。**本阶段只产出提示词，不调用任何生图接口，不生成真实图片。**

## 一、设计师视角：朋友圈配图方法论

朋友圈配图要的是"像本人随手拍的真实感"，不是精修大片，也不是一眼就能看出来的 AI 塑料感。设计前先问自己：

- 这条文案想传达的情绪/态度是什么（松弛 / 自嘲 / 治愈 / 凡尔赛……），画面要和文字的态度对上，不能各说各话。
- **默认只出 1 张图**；只有用户明确要组图（如探店、旅行）才出多张，且每张切一个不同角度/细节，不是同一场景的重复构图。
- 画面感要"生活化"而不是"广告感"：不需要完美构图，可以带一点真实手机随拍的视角/景深/构图随意感，除非用户明确想要"高级简洁风"这种偏精致的调性。

## 二、每张图要确定的信息

| 维度 | 说明 |
|---|---|
| 画面主体 | 具体是什么（场景 / 物品 / 局部细节；朋友圈很少用人物正脸） |
| 氛围与视角 | 手机随手拍视角为主；氛围感 vs 精致感由人设风格决定 |
| 色调与光影 | 与文案情绪匹配的主色调 |
| 风格关键词 | 3–5 个 |
| 比例 | 见第五节参考比例（朋友圈默认竖版或方图） |

## 三、风格方案：在人设风格内给 2–3 套细分方向供选择

朋友圈配图的大方向已经由阶段 2 用户选定的人设风格（`references/styles/`）决定，不用重新问一遍"想要什么风格"；但同一人设风格下仍有不同的"分寸感"可选，出提示词前先给 2–3 套细分方案，让用户挑一套贴近当下心情的：

| 人设风格 | 细分方案示例 |
|---|---|
| 高级简洁风 | A. 纯留白极简构图 / B. 单一主体特写 |
| 幽默自嘲风 / 种草安利风 | A. 生活随手拍感 / B. 略带构图感但不精修 |
| 文艺emo风 | A. 低饱和冷调氛围 / B. 暖光颗粒感氛围 |
| 凡尔赛风 | A. 极度日常化不起眼 / B. 留一点耐人寻味的细节 |
| 温暖治愈风 | A. 局部细节特写 / B. 大场景暖光氛围 |

> 以上是示例细分方向，可根据文案实际情绪调整或另拟，但**必须给出至少 2 套供用户挑选，不能替用户直接拍板一套了事**。**默认只设计 1 张图**，用户明确要组图才设计多张，每张切不同角度/细节。

## 四、中文出图提示词公式（通用，适配主流 AI 生图工具）

```
[画面类型：朋友圈随拍照片] + [画面主体与场景] + [氛围与视角] + [色调与光影] + [风格关键词] + [画面比例] + [负面提示]
```

示例（深夜 emo，文艺emo风 · 细分方案 A 低饱和冷调）：

> 一张朋友圈随手拍照片，画面主体是深夜便利店门口的塑料椅子和一杯没喝完的热美式，手机平视随拍视角，微微失焦的背景是店内暖黄灯光。色调低饱和、偏冷蓝夜色与暖黄灯光的对比，氛围安静克制。风格：真实生活质感、轻微颗粒感、非摆拍构图。竖版构图。避免：摆拍感过强、过度精修、卡通感、水印、文字。

负面提示统一建议包含：摆拍感过强、过度精修、卡通感、水印、文字乱码、与设定色调不符的杂色。

## 五、参考比例

| 用途 | 建议比例 |
|---|---|
| 方图（朋友圈九宫格单图） | 1:1 |
| 竖版图（朋友圈单图默认） | 3:4 |
| 横版图（风景/场景类） | ~16:9 |

> 具体像素/参数格式因生图工具而异（如 Midjourney 用 `--ar 3:4`，即梦 / Stable Diffusion 各有自己的尺寸选择器），提示词末尾给出比例文字即可，由用户在所用工具里选择对应挡位。

## 六、交付方式

- 提示词以完整、可直接复制粘贴的一段文字呈现（含负面提示、比例建议）；本阶段只在对话中展示，不主动调用任何生图接口或生成图片文件。
- 展示提示词后，用一句话简述画面重点，方便用户核对是否符合预期。
- 用户想调整 → 换一套细分方案重新出提示词，或只微调某个细节（换主体/换色调/加减风格词），直到满意为止。
- 用户确认满意后，选定的细分方案与提示词会随文案定稿一起，在阶段 7 落盘写入 `doc/` 目录（见 SKILL 文档「输出约束」），不需要在本阶段单独保存。
