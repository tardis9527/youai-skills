# UI/UX 设计风格重塑

> 🍊 来自 [YouAI Skills](https://github.com/tardis9527/youai-skills) — 为创造者准备的 AI 技能包

## 使用方法

请完整阅读本目录下的详细 Skill 定义文件 `../../05_uiux-redesign.md`，按照其中定义的角色设定、执行步骤和报告结构执行分析。

## 核心流程

1. **产品上下文理解**：采集产品背景、商业目标、用户群体画像，建立设计决策的底层逻辑
2. **现状设计审计**：扫描项目代码中的 UI 实现，系统性评估视觉与体验问题
3. **设计策略定义**：确定设计调性、制定设计原则、提出设计方向提案
4. **设计规范制定**：输出完整的配色、排版、间距、组件、交互规范
5. **实施方案规划**：Design Token 方案、组件改造计划、分阶段路线图
6. **输出设计重塑方案**：整合为完整交付文档

## 方案结构

1. 设计背景与目标（产品定位、优化目标、用户画像）
2. 现状审计摘要（关键问题 Top 5、审计评分）
3. 设计策略（调性定位、设计原则）
4. 设计规范（配色、排版、间距、组件、动效）
5. 实施方案（Token 方案、组件改造、路线图）
6. 风险与降级策略
7. 附录（竞品参考、设计决策记录）

## 约束

- 所有设计决策必须回溯到用户群体和产品目标
- 必须基于项目代码审计，不能脱离现有技术栈
- 使用中文输出
- 保存至 `doc/UI_UX_设计重塑方案_{产品名称}_{YYYYMMDD}.md`
