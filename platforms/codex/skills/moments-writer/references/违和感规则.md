# 朋友圈违和感规则

朋友圈文案最容易露馅的不是"说明书腔"，而是**塑料感**——一眼看出是文案/鸡汤号/营销号写的，不是朋友自己发的。写完先跑这套规则清洗。

## 板块一 · 朋友圈滥用套话（发现即删或重写，不许保留）

- **鸡汤开场腔**："生活明朗，万物可爱""愿你眼中有光，心中有爱""岁月不居，时节如流"
- **烂梗体开头**："入职三天体"("入职第 3 天，感觉自己是废物")、"人间不值得"体、"上一秒……下一秒……"强行转折体
- **营销号收尾**："转发给需要的人""致敬每一个努力生活的自己""与朋友们共勉"
- **空洞抒情词**：岁月静好、诗和远方、不负韶华、余生很长、山高水长（除非用户明确要这种调性）
- **AI 味强行金句**：结尾强行升华出一句"人生哲理"，尤其是原本只是晒张吃饭照

## 板块二 · emoji 使用规则

- 同一句 ≤ 3 个 emoji，禁止一句话后面挂 5、6 个 emoji 的"营销号刷屏感"
- emoji 要贴场景，不能为了凑数硬加（如认真吐槽职场压力时插一堆🎉🌈会显得违和）
- 用户没提要求时，emoji 数量按人设风格默认值来（见 `styles/` 各风格文件），不要自作主张往多了加

## 板块三 · 篇幅红线

- 朋友圈不是公众号，**默认不写小作文**，除非用户明确要"小段落/情绪长文"
- "一句话文案"档：控制在一句话内，不分段
- "短句+emoji"档：2-4 句，可配 emoji，不写大段背景交代
- "小段落"档：最多 3-4 行，仍要留白，不要写成完整叙事文章

## 板块四 · 真实感检查

- 全文要像"这条朋友圈是本人发的"，而不是"AI 代写的通稿"
- 允许错别字风的口语（"绝了""破防了""救命"）、允许不完整句、允许突然的语气词
- 忌通篇工整对仗、忌每条候选都用同一个句式模板（如全都是"当……的时候，才……"）
- 忌堆砌网络热梗到失真（一条里塞 3 个以上热梗容易显得刻意）

## 板块五 · 最终检查清单

1. 是否清掉了鸡汤开场腔 / 烂梗体 / 营销号收尾？
2. emoji 是否 ≤ 3 个/句，且贴合场景？
3. 篇幅是否符合用户要求的档位（一句话/短句/小段落）？
4. 读起来像不像本人发的，而不是代写通稿？
5. 多条候选之间是否用了不同角度/句式，而不是同一模板换词？
