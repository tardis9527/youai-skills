# 轻量检查表

朋友圈文案篇幅短、产出快，不需要公众号那套六维长文评分。每条候选写完后，过这张轻量检查表即可，**不通过就直接重写那一条**，不做多轮打分。

## 四项一票否决（任一不满足，重写该条）

1. **无朋友圈滥用套话**：对照 `违和感规则.md` 板块一，无鸡汤开场腔 / 烂梗体开头 / 营销号收尾
2. **emoji 不过量堆砌**：同一句 ≤ 3 个 emoji，且未经用户要求不强行加
3. **篇幅符合要求档位**：一句话 / 短句+emoji / 小段落三选一，没有超范围写成小作文
4. **人设语气一致**：与阶段 2 选定的风格文件（`styles/`）的语气、句式、标志性表达对得上

## 候选组内检查（针对一次生成的 3-5 条）

- 是否覆盖了不同角度（如同一件事，从"吐槽"“凡尔赛”“云淡风轻"三种角度切入），而不是同一句话换几个词
- 是否至少有 1 条明显更短、1 条略带态度/情绪，给用户实际的选择空间
- 是否都符合用户这次要求的核心信息（不能为了"有梗"而偏离用户想表达的事实/情绪）

## 输出格式（自查时用，不必展示给用户）

```
【轻量检查】
候选1：✔/✘ 套话  ✔/✘ emoji  ✔/✘ 篇幅  ✔/✘ 人设一致
候选2：……
结论：N 条通过，M 条已重写
```

自查后直接把通过的候选交给用户挑选即可，不需要把检查表本身展示出来。
