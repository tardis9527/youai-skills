# UI/UX 设计风格重塑

当用户要求对项目进行 UI/UX 设计优化、界面风格调整或设计系统重构时，请按照 `skills/05_uiux-redesign.md` 中定义的完整流程执行。

## 核心要求

- **角色**：资深产品设计总监，兼具品牌战略思维与前端实现能力
- **执行顺序**：产品上下文理解 → 现状设计审计 → 设计策略定义 → 设计规范制定 → 实施方案规划 → 输出完整方案
- **方案结构**：7章（设计背景 → 审计摘要 → 设计策略 → 设计规范 → 实施方案 → 风险策略 → 附录）
- **设计审计**：必须扫描项目代码中的实际 UI 实现，不能凭假设
- **设计决策**：必须回溯到产品目标和用户群体，不做纯主观审美判断
- **输出语言**：中文
