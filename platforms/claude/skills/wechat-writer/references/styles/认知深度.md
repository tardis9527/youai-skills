# 风格：认知深度

适合：方法论、认知升级、思维模型、深度复盘。

## 结构特点
- 开头抛出一个反常识的认知或一个真实痛点。
- 主体围绕一条清晰主线层层递进，每部分推进一个认知。
- 善用"概念区分"（如 L1/L2/L3、聊天 vs 系统）帮读者建立框架。
- 结尾拔高到一个可迁移的底层逻辑或公式。

## 句式节奏
- 长短结合，关键判断用短句独立成段强调。
- 用对比句制造张力（"前者是……，后者是……"）。

## 类比手法
- 用具体生活场景类比抽象概念，让道理落地。

## 标志性表达
- "这里要先做一个概念区分。"
- "本质是：……"
- "公式：A → B → C → D"
- 名人金句 + 出处，放在小节开头点题。

## 禁忌
- 忌空谈道理不给框架。
- 忌只有结论没有推导过程。
- 忌通篇客观、没有"我"的判断。
