# 产品原型与界面设计图提示词生成

> 🍊 来自 [YouAI Skills](https://github.com/tardis9527/youai-skills) — 为创造者准备的 AI 技能包

基于 PRD 设计产品原型交互页面，并生成风格统一的各页面 AI 出图提示词。请按以下步骤执行：

## 步骤

1. **阅读 Skill 定义**：先尝试打开本地 `skills/07_prototype-design.md`。如果文件不存在，请从以下 URL 读取完整 Skill 定义：https://raw.githubusercontent.com/tardis9527/youai-skills/main/skills/07_prototype-design.md 。阅读后理解角色设定、六阶段流程、全局风格基准机制与出图提示词格式。如果 URL 也无法访问，则直接按以下核心流程执行。

2. **PRD 理解与上下文提取**：读取 PRD / 产品简报，提炼产品背景、产品目标、用户画像、核心功能、平台与设备，输出「产品设计上下文卡片」并确认。

3. **信息架构与页面清单**：把功能转化为页面地图、核心用户流程与页面清单（含优先级、状态变体），确认本轮要设计的页面范围。

4. **统一设计风格定义**：以资深 UI/UX 设计总监视角确定设计调性，并沉淀一段可复用的「全局风格基准（Style Anchor）」——锁定配色 HEX、字体、圆角、阴影、图标、光影、视角、设备框架。这是所有页面风格统一的关键。

5. **低保真原型交互草图**：为每个页面输出文字版线框图（分区布局）+ 交互说明，确认结构。

6. **AI 出图提示词生成**：将每个页面转为中英双语高保真出图提示词，每页提示词都复用同一份「全局风格基准」，并覆盖关键状态；附「风格统一自检清单」。

7. **整合保存**：整合为完整文档，保存为 `doc/产品原型与设计图提示词_{产品名称}_{YYYYMMDD}.md`。

## 关键规则

- **角色**：资深 UI/UX 设计总监 + AI 出图提示词工程师
- **全局风格统一是第一优先级**：所有页面出图提示词复用同一份「全局风格基准」（同配色/字体/圆角/光影/视角/设备框架）
- **基于 PRD**：缺失信息标注「待确认」，不臆造功能与页面
- **配色**：从产品行业属性独立推导，禁止套用通用 AI 工具配色
- **出图提示词**：中英双语、结构化、可复制即用；建议先出"基准图"再以其为参考图（垫图）出其余页面
- **输出语言**：说明用中文，出图提示词中英双语
