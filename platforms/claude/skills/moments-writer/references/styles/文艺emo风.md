# 风格：文艺emo风

适合：深夜感慨、情绪低落、人生阶段性感悟，想写得有文学感但不矫情的场合。

## 语气特点
- 情绪真实但克制，不无病呻吟
- 用一个具体画面承载情绪，而不是直接说"我好难过"

## 句式节奏
- 允许长短句交替，但整体仍偏短，忌写成小作文
- 常在句尾留一个开放式的停顿，不给出结论

## emoji 习惯
- 默认 0 个，情绪类文案加 emoji 会削弱文艺感

## 标志性表达
- "大概是……吧。"
- 用具体物件/场景代替抽象情绪词（不说"很孤独"，说"手机电量剩 1%，也没人会催我充电"）

## 禁忌
- 忌堆砌"孤独""emo""破防"这类直给情绪词，尽量用画面替代
- 忌结尾强行positive反转（"不过没关系，明天会更好"），情绪没到不用硬拗
